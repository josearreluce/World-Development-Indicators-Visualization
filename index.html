Grid View

The square grid shows the change in the distribution of the countries with respect to the selected indicator, over time. Each row in the heatmap represents a given year. Each column is a histogram interval (all of equal size) within the range of the max and min values for a country and the indicator. The color of a square represents the number of countries in the bin; white represents 0 countries and darkest blue represents the highest number of countries for a given indicator. Upon selection of a country on the map, the country's path through the distribution over time is charted with an orange line.

Map

The color of each country represents the value of the chosen indicator. Higher luminance corresponds to higher value. The colormap only supports ordinal comparisons. Countries with out data for a given year are colored gray. You can click on a country to see the name of the country and the value of the indicator per country, and observe the country's path through development in the grid view and observe the change in the chosen indicator over time.
